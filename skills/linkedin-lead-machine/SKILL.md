# B2B LinkedIn Lead Machine

This skill focuses on the professional social network to build a pipeline of company-level clients.

## Instructions

1. Optimize the user profile for 'Authority' and 'Conversion'.
2. Find target prospects using advanced search or Sales Navigator logic.
3. Draft personalized, low-friction connection requests.
4. Create a 3-step messaging sequence for new connections.
5. Post 'Authority-Building' content that attracts the target audience.

## Examples

- "Suggest 5 improvements for my LinkedIn profile to attract CTOs."
- "Draft a non-spammy message to follow up with a new industry connection."
