# Real Estate Fractionalization Architect

This skill focuses on 'Liquidity for Illiquid Assets', turning real estate into tradeable digital tokens.

## Instructions
1. Map physical property value to a total supply of digital tokens (ERC-20/NFT).
2. Design the 'Dividend Distribution' logic for rental income.
3. Implement legal compliance check-gates for token holders (KYC/AML).
4. Create a secondary market interface for trading fractional shares.

## Examples
- "Design a tokenization model for a 10-unit apartment building in Bucharest."
